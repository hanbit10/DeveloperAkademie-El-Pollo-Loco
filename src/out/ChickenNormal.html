reset() → {void}

Resets the state of the chicken to its initial values. This function sets the `deadSetting` flag to `false`, and resets the `x`, `y`, `speed`, and `energy` properties to their initial values. The `x` and `y` properties are reset to their cached values, and the `speed` property is set to a random value between 0.55 and 2.55. The `energy` property is set to 100.
