animate() → {void}

Animate the chicken's movement and actions at regular intervals. This function uses `setInterval` to repeatedly execute a callback function at a specified interval. The callback function checks if the chicken is not dead using the `isDead()` method. If the chicken is not dead, it calls the `playAnimation()` method with the `IMAGES_WALKING` array as an argument.
